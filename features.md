# Feature Roadmap

## Done

- [x] Self-rating after card flip — Show "Got it" / "Missed it" buttons after revealing the answer. Display a score summary at the end and offer to re-study only missed cards.
- [x] Keyboard navigation — Space to flip, arrow keys to navigate, 1/2 for got it/missed it. Essential for efficient studying.
- [x] Deck library with local persistence — Save decks to localStorage so returning users see their previous decks without needing a share link.

## Planned

- [ ] Edit cards before studying — Tap a card in the preview list to edit front/back text. Useful for fixing AI-generated or CSV typos.
- [ ] Anki export — Export decks to Anki-compatible format (.apkg or Anki CSV/TSV) for students who use Anki as their primary tool.
- [ ] Quiz mode — Type-the-answer mode with fuzzy matching against the card back for more active recall.
